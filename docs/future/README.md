# Future And Prototype Surfaces

This directory is the quarantine area for concepts that are not part of the
testnet kernel in `docs/TESTNET.md`.

Current future/prototype-only areas:

- historical tokenfactory lifecycle material;
- historical native DEX smoke and lifecycle material;
- native market, auction, token, or NFT application-asset module ideas;
- AVM standards that are present as specifications or helper packages but are
  not yet deployed and exercised as contracts.

Launch docs must describe these areas as future AVM contracts or standards, not
as active native application-asset modules.
